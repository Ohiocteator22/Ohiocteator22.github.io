Roblox UI Framework

Mobile-first UI,
built kernel by kernel.

CornUi is a universal Roblox UI framework from Ohiocteator22 — built for phone and PC alike, with the simplicity of Orion and none of the UNC compiler headaches.

Get started View repo ↗ Redirect to Plugins API
-- load the library
local Corn = loadstring(game:HttpGet('https://raw.githubusercontent.com/Ohiocteator22/Corn-Ui-Reloaded/refs/heads/main/Sourse/CornUi.lua'))()

-- spin up a window
local Window = Corn:CreateWindow({
  Name = 'Corn Hub',
  Icon = 80406291512141
})
What's inside

A full toolkit, not just a window and a button.

Everything scales automatically between phone and desktop, and every element ties into the same flag and theme systems.

Mobile-first scaling

Auto-scaling layout tuned for touch first, desktop second.

Theme editor

Live single-key recoloring across the whole UI, tween-free.

Command palette

Embedded in the title bar, with keyword + argument parsing.

Flag manager

Shared flag table with get/set and a FlagChanged event.

Search

Matches name, keywords, and description across every element.

Notifications

Color-coded toasts with a countdown bar.

Spotlight & backgrounds

Image and video backdrops, plus element spotlighting.

Plugin API

Sandboxed plugin loading via a pcall-wrapped ctx closure.

Why CornUi

Built for the executor you're actually running.

1 script

Same source runs on mobile and PC, no separate builds.

0 UNC limits

loadstring workaround sidesteps line-length caps on strict compilers.

Live

Theme editor recolors instantly, no tween, no reload.

Shipping

Actively developed — plugin API and new widgets land regularly.

Orion-simple. Built for phones first.

If your last UI lib fought you on mobile, CornUi is the fix — same ease of use, none of the desktop-only assumptions.

Install

Two lines to a working window.

Swap in your own raw script URL once it's hosted, then build out tabs and sections from there — full API in Documentation.md and Widgets.md.

local Corn = loadstring(game:HttpGet('https://raw.githubusercontent.com/Ohiocteator22/Corn-Ui-Reloaded/refs/heads/main/Sourse/CornUi.lua'))()
local Window = Corn:CreateWindow({ Name = 'Corn Hub' })

Official Plugin

Performance Suite

Performance-related tooling and options for CornUi, loaded through the standard plugin API.

Load it

Performance Suite loads like any other CornUi plugin, through Library:LoadPlugin with a raw URL pointing at the plugin's source file.

-- after Corn and Window are set up
Corn:LoadPlugin('https://raw.githubusercontent.com/Ohiocteator22/Corn-Ui-Reloaded/refs/heads/main/Plugins/DevSuite.lua')

Notes
